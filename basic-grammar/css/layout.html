布局要点

  • 用行和列去构思：从左至右或从上至下
  • 用矩形边框去划分页面
  • 使用语义化的html元素

行内元素和块级元素

  • 行内元素：span、button、img
  • 块级元素：article、div、ul、li、p

边距

  • 在元素右侧和下方设置 margin，不去碰左侧和上方的 margin。
  • CSS 在竖直方向上有 margin 坍塌现象。
